# MiseBuy brand system

Reading this as: a micro-SaaS landing for independent restaurant owners and chef-operators, with a pragmatic editorial-operations language, leaning on native CSS, a humanist system sans, and industrial kitchen material cues.

- `DESIGN_VARIANCE: 7/10`
- `MOTION_INTENSITY: 5/10`
- `VISUAL_DENSITY: 4/10`

## Concept

Two supplier columns converge into one checked buying list. The mark reads as an `M`, two price sheets, and an open receiving tray.

## Voice

- Concrete
- Operator-respectful
- Calm under margin pressure

## Tokens

- Cool off-white: `#F3F1EA`
- Ink: `#20211E`
- Tomato accent: `#D45138`
- Stainless gray: `#A7AAA3`
- Soft radius: 14px on surfaces, pill buttons, 9-10px fields
- Type: system humanist sans and system mono for prices

## Motion rationale

- Hero entrance establishes hierarchy.
- Result transition confirms state change.
- Skeleton communicates comparison progress.
- Button press acknowledges action.
- All motion collapses under `prefers-reduced-motion`.

The generated board is `brand-board.png`. The production mark was reduced to a simple code-native SVG for deterministic rendering at 16px.
